---- Procedimentos para executar o programa -----

1 - Executar o pyro-ns
2 - Executar todas as classes distribuidas, que são [Joke.py, Proverb.py, Maintenance.py] 
3 - Agora execute o servidor [MainServer.py]
4 - Execute o client.py e o admcliente.py
   
  4.1 - client.py
    Aqui o cliente  escreve seu nome e dependendo do modo do servidor você recebe [piada, proverbio ou manutenção]
 
  4.2 - admclient.py
    Aqui o administrador digita o modo no qual deseja que fique o servidor


The End :)
